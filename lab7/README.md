## Запуск проекта

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/MatoiDev/AlgoLabs.ITMO.git
   ```

2. Перейдите в папку с любой лабораторной работой:
   ```bash
   cd AlgoLabs.ITMO/lab<N>
   ```

3. Запустите программу:
   ```bash
   python run_tasks.py
   ```
